# Changelog

All notable changes to the `session-handoff` plugin are documented here.

## [0.1.0] - 2026-07-09

### Added
- Initial release: writes and resumes structured context handoffs between Claude sessions —
  trust-tagged claims (`[V]`/`[?]`/`[S]`), parent/chain tracking, working-tree-aware staleness
  checks, referenced-file validation, and a stdlib-only helper script with a smoke test.
